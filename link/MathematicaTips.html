Mathematica Tips

1. Follow these tips in the Wolfram blog.

2. Try to parallelize the computation.

3. Try a remote login to the Faculty server. (username@staff.science.uu.nl in the case of ITF Utrecht)

4. Use a large number of CPUs (a workstation cluster).

Usually one cannot open a notebook, so s/he needs to send jobs to the Mathematica kernel. For this purpose, a notebook should be rewritten a batch file.

See How to use Math kernel.

5. To save the current notebook, use NotebookSave[EvaluationNotebook[]]. You do not have to go in front of the desktop and click "Save".

6. To clear all memories in the kernel and keep evaluating the cells, use SetOptions[$FrontEnd, "ClearEvaluationQueueOnKernelQuit" -> False]; Quit[].
