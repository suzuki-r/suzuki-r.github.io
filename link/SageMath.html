1. Introduction

SageMath is an open-source mathematical software, similar to Wolfram Mathematica. SageMath has an interface to GAP by default.

2. Install

For Mac OSX, a simple way is to download a binary release (e.g. SageMath-10.2_arm64.dmg).

For Linux, it is a bit complicated to install the latest version. The best way is to follow the conda-forge instruction. 

curl -L -O https://github.com/conda-forge/miniforge/releases/latest/download/Mambaforge-$(uname)-$(uname -m).sh

Grant permission to this Mambaforge file using chmod, and execute it, 

./Mambaforge-$(uname)-$(uname -m).sh

You may need to add PATH in the usual way by rewriting ~/.bashrc 

export PATH=$PATH:/home/(something)

Install sage using Mamba, 

mamba create -n sage sage python=3.10

Mamba is needed because it takes a long time if conda is used. Mambaforge is needed, because if you may not be able to install mamba on anaconda in Linux due to conflicting packages. If you try the standard Linux Pacakage Manager, 

sudo apt-get install sagemath

this installs an older version.

3. Launch

If you used Mamba to install sage, then type 

mamba activate sage
jupyter notebook

The rest of procedures is similar to pyenv.

4. Basics

In order to use the functions of library such as GAP, add the library name, like 

S2S2 = gap.DirectProduct(SymmetricGroup(2), SymmetricGroup(2))
orbits = gap.OrbitsDomain(S2S2,SymmetricGroup(4))

The contents of a variable can be displayed by print(), show(). The function view() gives LaTeX output as a PDF. 

show(orbits)

You can reuse the last output by _, the second last by __, the thir last by ___, and the n-th last by _n. Note that the definition of a function produces no output.

5. Tips

When you run a SageMath code, the execution at Command Prompt is much faster than opening a notebook; see here. Similarly, GAP functions are much faster than using Python functions.
